> ### Keterangan
>
> - ``geo12vero = ID Mapel``
> - ``0geo12vero = Tipe Soal + ID Mapel = 0 + geo12vero``
>
>> **Tipe Soal**
>> - ``0 = Untuk Soal Biasa/Umum atau PG``
>> - ``1 = Untuk Soal MTK PG atau Essay``
>> - ``2 = Untuk Soal Listening Inggris``
>> - ``4 = Untuk Soal PG``
